# Ride With Foxy

A mobile-friendly private transportation booking MVP for Las Vegas.

## Included
- No customer login
- One-way, round-trip, airport, hourly and full-day options
- Instant estimate
- One-tap text booking request
- Responsive blue-and-black design

## Before launch
Replace the placeholder phone number `17025550123` in `index.html` with the business booking number.

## Publish with GitHub Pages
Open **Settings → Pages**, choose **Deploy from a branch**, select **main / root**, then save.
